# JSON Helper com NodeJS

## CRUD simples com a lib File System
Desenvolvido para facilitar a manipulação de arquivos JSON nas atividades diárias com JS.

## FUNÇÕES
- Criar
- Ler
- Atualizar conteúdo
- Deletar
